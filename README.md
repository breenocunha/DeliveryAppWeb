### Configuração do Backend

#### Entrar na pasta
`cd backend`
#### Instalar dependências
`npm install`
#### Configurar variáveis de ambiente
`cp .env.example .env`
#### Executar migrations
`npm run migrate`
#### Iniciar servidor
`npm run dev`

### Configuração do Frontend

#### Entrar na pasta
`cd frontend`
#### Instalar dependências
`npm install`
#### Iniciar aplicação
`npm start`
